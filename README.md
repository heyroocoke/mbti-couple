# MBTI 궁합 테스트

무료 MBTI 성격 유형 검사와 연인 궁합 테스트 웹 애플리케이션입니다.

## 주요 기능

- **MBTI 테스트**: 20문항으로 나의 MBTI 유형 알아보기
- **궁합 테스트**: 16가지 유형별 커플 궁합 확인
- **결과 공유**: 카카오톡 공유 및 링크 복사 기능
- **다국어 지원**: 한국어, 영어, 일본어, 중국어 자동 감지

## 다국어 지원 (i18n)

브라우저 언어 설정에 따라 자동으로 해당 언어로 표시됩니다.

### 지원 언어
- 🇰🇷 한국어 (Korean)
- 🇺🇸 영어 (English)
- 🇯🇵 일본어 (Japanese)
- 🇨🇳 중국어 간체 (Chinese Simplified)

### 기술 스택
- `i18next` + `react-i18next`
- `i18next-browser-languagedetector` (자동 언어 감지)

사용자는 우측 상단의 언어 선택 버튼으로 수동 변경도 가능합니다.
